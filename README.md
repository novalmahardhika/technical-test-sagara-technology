## How to run this code ?

### Clone repo

```bash
git clone  https://github.com/novalmahardhika/technical-test-sagara-technology.git
```

### Go directory

```bash
cd technical-test-sagara-technology
```

### Install package

```bash
npm i
```

### Run app

```bash
npm run dev
```
